# Saq-Form

Extension to the [Saq](https://github.com/Piasiu/saq) framework for form support.

# Installation

It's recommended that you use [Composer](https://getcomposer.org/) to install Saq-form.
```bash
$ composer require piasiu/saq-form
```
This will install Saq-Form and all required dependencies. Saq-Form requires PHP 8 or newer.
